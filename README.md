# FullstackFighters

Our goal is to build a dual player fighting game implementing machine learning to allow for an interactive fighting experience.

1.npm install

2.npm start

3.running on localhost:7777/

4.Enjoy

# Controls

King:
Up: "w"
Left: "a"
Right: "d"
Attack: "space"
Special Attack: "x"
Shield: "j"

Ghost:
Up: "arrow up"
Left: "arrow left"
Right: "arrow right"
Attack: "arrow down"
Special Attack: "m"
Shield: "n"
